# Nocturne Scholastic — Design System

## Brand & Style

Dark Academic + Modern Corporate. Atmosfera da biblioteca digitale.

## Colori

| Ruolo | Colore |
|---|---|
| Sfondo principale | `#13131b` |
| Superficie card | `#1f1f27` |
| Superficie alta | `#34343d` |
| Primario | `#c0c1ff` |
| Secondario | `#ddb7ff` |
| Terziario | `#ffb783` |
| Testo su superficie | `#e4e1ed` |
| Bordo | `#908fa0` |

## Tipografia

| Stile | Font | Peso | Dimensione |
|---|---|---|---|
| Display | Inter | Bold 700 | 48px |
| Headline | Inter | SemiBold 600 | 32px |
| Titolo | Inter | SemiBold 600 | 20px |
| Body | Inter | Regular 400 | 16px |
| Label | Inter | Medium 500 | 13px |
| Codice | JetBrains Mono | Regular 400 | 14px |

## Spaziatura

Base 8px — tutti gli incrementi sono multipli di 8.

| Token | Valore |
|---|---|
| `xs` | 4px |
| `sm` | 12px |
| `md` / gutter | 24px |
| `lg` | 40px |
| `xl` | 64px |

## Arrotondamenti

- **Componenti base** (bottoni, input): 8px
- **Contenitori** (card, pannelli): 16px
- **Checkbox**: 4px

## Elevazione

3 livelli: Base (`#13131b`) → Superficie (`#1f1f27`) → Popover (`#34343d`).

## Componenti chiave

- **Bottoni primari**: gradiente indaco-viola, testo bianco
- **Bottoni secondari**: bordo 1px, sfondo trasparente
- **Card**: sfondo `#1f1f27`, bordo `#334155`, padding 24px
- **Input**: sfondo più scuro del contenitore, focus con bordo indaco e glow
- **Chips/Tag**: colore accent desaturato al 20% di opacità
